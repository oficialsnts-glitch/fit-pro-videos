# Biblioteca de Vídeos - SDR Pro

Repositório contendo a biblioteca de vídeos de exercícios para o aplicativo SDR Pro.

Os vídeos são servidos via GitHub Raw Content (CDN gratuito) para manter o PWA leve e rápido.

## URLs de Acesso

Os vídeos podem ser acessados em:
```
https://raw.githubusercontent.com/oficialsnts-glitch/fit-pro-videos/main/videos/{nome_do_arquivo}.mp4
```

## Estrutura

- `videos/` - Pasta contendo todos os vídeos de exercícios em formato MP4
